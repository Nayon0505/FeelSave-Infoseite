Tech Stack von FeelSave - Native Android App

Zum Git Repository

Entwicklungsumgebung & Programmiersprache

Android Studio: Die offizielle IDE zur Android-Entwicklung bietet umfassende Tools für Design, Coding, Debugging und Performance-Optimierung.

Programmiersprache: Das Projekt wird in Java oder Kotlin entwickelt - beides moderne Sprachen, die eine performante und wartbare App ermöglichen.

Android SDK & Bibliotheken

Android SDK: Bietet alle nötigen APIs und Tools für die Interaktion mit Android-Geräten - von UI-Komponenten bis hin zu Systemdiensten.

AndroidX & Material Design: Moderne Bibliotheken, die ein konsistentes, benutzerfreundliches und optisch ansprechendes Design ermöglichen.

Drittanbieter-Bibliotheken: Tools wie Retrofit (für Netzwerkanfragen), Glide oder Picasso (für das Laden und Cachen von Bildern) unterstützen zusätzliche Funktionalitäten.

Build System & Versionskontrolle

Gradle: Das Standard-Build-System für Android automatisiert den Kompilierungs-, Test- und Deployment-Prozess.

Git & GitHub: Für Versionskontrolle und kollaborative Entwicklung wird Git eingesetzt - das Repository liegt auf GitHub, was eine strukturierte Zusammenarbeit ermöglicht.

Zusätzliche Services & Backend-Integration

Firebase (optional): Firebase kann zur Authentifizierung, Datenbankverwaltung (Realtime Database oder Firestore) und für Push-Benachrichtigungen integriert werden.

RESTful APIs: Die App kann externe APIs konsumieren - dies wird typischerweise mit Bibliotheken wie Retrofit umgesetzt, um zusätzliche Dienste oder Daten einzubinden.
